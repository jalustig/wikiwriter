## Available Task Types

**research_section** — Extract claims from a section and find sources for uncited ones.
  params: {{"section": "<section name>"}}
  Use before draft_section when edit_type is CITE_REPAIR, EXPAND, or FACT_CHECK.
  Do NOT use before PRUNE sections (pruning doesn't need new sources).

**draft_section** — Write an improved version of a section.
  params: {{"section": "<section name>", "mode": "CiteFix|Expand|FactCheck|Prune"}}
  mode mapping: CITE_REPAIR→CiteFix, EXPAND→Expand, FACT_CHECK→FactCheck, PRUNE→Prune
  Must list all research_section tasks for this section as deps.

**synthesize** — Assemble all section drafts into a coherent article.
  params: {{}}
  Must be the final task. deps: all draft_section tasks.

## Constraints
- effort_ceiling: {effort_ceiling}

  - LIGHT: draft only, no research; limit to 2 sections max
- Only include sections with action=EDIT from the assessment
- Always end with exactly one synthesize task
- Keep IDs short: t1, t2, t3, ...
